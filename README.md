# Glovo-sales-order-sql-analysis
  PROJECT OVERVIEW

This project analysis a glovo order dataset using SQL to uncover valuable business insights relating to sales performance, profitability customer behaviour, delivery efficiency and payment trends. This analysis demonstrate how SQL can be used to answer real world business questions and support data driven decision making.

  PROJECT OBJECTIVES
- It analyze sales and profit performance across different locations.
- Evaluate customer and purchasing behaviour and profitability.
- Measure delivery performance and customer satisfaction.
- Compare payments method based on revenue generation.
- Identify high value customers using SQL analytics.

    BUSINESS QUESTIONED ANSWERED
1. Which top 10 areas generated the highest revenue?
2. Which customer segment has the highest average order value and total profit?
3. For each month, what were the total order, revenue and profit?
4. Which payment method contributes the highest total revenue and average order value?
5. Which areas have an average delivery time above the overall average delivery time?
6. Rank customers segment by total profit (using SQL ranking functions)?
7. Which delivery partners handled the most orders and what was their average delivery time and average customer rating?
8. How does customer rating differ between on-time and delayed deliveries?
9. Which customers have placed more orders than the average customer and generated above revenue?

    SQL SKILLS DEMONSTRATED
- Aggregated functions (SUM,AVG,COUNT)
- GROUP BY
- ORDER BY
- CASE WHEN
- Common table expressions (CTEs)
- Window functions (RANK())
- Subqueries
- Date functions
- Filtering and sorting.

    KEY BUSINESS INSIGHTS
- It identified the most profitable areas and customers segments.
- It measured delivery performance and its impacct on customer ratings.
- It evaluuated payment methods based on revenue and order value.
- It tracked monthly sales, revenue and profit trends.
- It identifitd high value customers using SQL analytics.

    CONCLUSION

  This project demonstrate how SQL can be used to transform raw transactional data into meaningful business insights. By analysing sales performance, customer bahaviour, delivery efficiency and profitability. The project showcases practical SQL techniques commonly used in data analytics. It highlights the ability to write efficient queries, solve real world business problems and support data driven decision making.
